# 🔐 Login System with Dashboard

## 📌 About
This project is a web-based login system that simulates user authentication and navigation between pages. It was developed to demonstrate front-end skills, user flow handling, and local data persistence.

---

## 🎯 Objective
Simulate a real login system with user validation and navigation flow between authentication and dashboard pages.

---

## 💡 Highlights
- User authentication simulation  
- Data persistence using localStorage  
- Navigation between pages (login → dashboard)  
- Logout functionality  
- Responsive and clean UI  

---

## 🚀 Features
- Email and password validation  
- Show/hide password  
- Error handling for invalid login  
- User session persistence  
- Protected dashboard (only accessible when logged in)  

---

## 🛠️ Technologies
- HTML  
- CSS  
- JavaScript
